Todoer is a personal ToDo app.

### Tech Stack

- Django
- Tailwind CSS

### Data

- Todo list item
  - content: plain text
  - created_at
  - marked_as_done_at

### User stories

- add a new entry
- mark entry as Done (checkbox)
- remove an entry

### Notes

This app does not have any notion of a User. Whoever visits it can do all the actions, there's no auth.
